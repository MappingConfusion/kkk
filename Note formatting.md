# Generic notes

In the markdown file, you must format your 'block' notes as follows (see [Inline notes](#inline-note-formatting) for notes on a single line):
<pre>
START
{Note Type}
{Note Fields}
Tags:
END
</pre>

To see how to format the Note Fields section, see [[Field formatting]]
